    Rozwiązania zadań ze Środowiska Programisty :

  • 1. Wyświetl na ekran 2 pierwsze wiersze pliku program.c.
    • head -n 2 program.c
  • 2. Wyświetl na ekran 4 ostatnie wiersze pliku program.c.
    • tail -n 4 program.c
  • 3. W pliku program.c znajdź wszystkie wiersze z wystąpieniem słowa „main”.
    • grep "main" program.c
  • 4. Plikowi program.c nadaj następujące uprawnienia: właściciel – czytanie, pisanie, grupa – czytanie, pozostali użytkownicy: brak uprawnień. 
    • chmod u+rw program.c
chmod g+r program.c
chmod o-rwx program.c
  • 5. Będąc w katalogu temp przenieś katalog wazne-sprawy do katalogu praca.
    • mv dom/wazne-sprawy praca/wazne-sprawy
  • 6. Zarchiwizuj cały katalog temp.
    • zip -r temp.zip temp
  • 7. Usuń katalog temp.
    • rm -R temp
  • 8. Odtwórz z archiwum katalog temp.
    • unzip temp.zip
  • 9. Posprzątaj na swoim koncie.
    • rm -r temp.zip

    Rozwiązania zadań z Wstępu do programowania:

    1. Dwumian Newtona
      #include stdio.h>
int newton(int n,int k){
        if(k==0||k==n) return 1;
        else return newton(n-1,k)+newton(n-1,k-1);
}
int main(){
        int n;
        int k;
        int wynik;
        printf("podaj liczbe n:");
        scanf("%d", &n);
        printf("podaj liczbe k:");
        scanf("%d", &k);
        wynik=newton(n,k);
        printf("silnia wynosi: %d", wynik);
        return 0;
}
    1. Objętość stożka
      #include stdio.h>
float stozek(float r,float h){
        return (3.14*r*r*h)/3;
}
int main(){
        float r;
        float h;
        float objetosc;
        printf("podaj wartosc promienia:");
        scanf("%f", &r);
        printf("podaj wartosc wysokosci:");
        scanf("%f", &h);
        objetosc=stozek(r,h);
        printf("Objetosc stozka wynosi: %f", objetosc);
        return 0;
}

if [ $1 -gt $2 ]
then

for i in *
do
mv $i $numer
numer=$[ numer + 1 ]
done
echo "Zmienionych plików: $numer"
